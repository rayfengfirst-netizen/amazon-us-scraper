# 发布目标约定

## 默认目标（当前）

- 服务器：`8.221.99.46`
- SSH：`root@8.221.99.46`
- 应用目录：`/opt/amazon-us-scraper`
- 服务名：`amazon-us-scraper`
- 监听：`0.0.0.0:8989`（见 `deploy/amazon-us-scraper.service.example`）
- **公网访问（当前生产入口）**：<http://8.221.99.46:8989/>
- eBay 列表：<http://8.221.99.46:8989/ebay>
- 健康检查：<http://8.221.99.46:8989/health>

说明：若未配置 Nginx 反代到 80 端口，则**必须带 `:8989`**；云安全组需放行 **TCP 8989**。

## 操作约束

1. 后续默认发布都走 `8.221.99.46`。  
2. 如需临时发布到其他机器，必须先在任务说明中明确。  
3. 不自动修改线上 `.env`；仅拉代码、装依赖、重启服务。
