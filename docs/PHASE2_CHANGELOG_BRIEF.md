# Phase 2 变更清单（简版）

适用对象：开发 / 运营 / 协作同学  
版本节点：`44e74fb`（后续增量见 2026-03-26 更新）

## 1) 页面与流程

- ASIN 详情页升级为 **Shopify 二次编辑后台**，可直接编辑后发布
- 原“传输预览”弱化，改为更贴近刊登后台的高效编辑体验
- 发布前可手动点击 **ChatGPT 改写四项**（标题、描述、SEO 标题、SEO 描述）
- 提交发布时不再自动 AI 改写，避免不可控耗时

## 2) Shopify 发布规则（当前）

- `vendor` 固定：`EGR Performance`
- `tags`：空字符串
- `SKU`：
  - Amazon：`AM-{ASIN}`
  - eBay：`EB-{item_id}`
- `price`：采集价 * `1.7`（可在页面继续编辑）
- 图片：使用高分图 URL 直传（不走本地 attachment）

## 3) 数据与采集策略

- 详情页读取数据库缓存结果，不依赖每次重新采集
- 图片下载策略收敛为：仅下载第一张高分图（降低资源开销）

## 4) 提示词库体系

- 新增提示词库配置文件：`prompts/shopify_openai/libraries.json`
- 每个模板包含四类提示词：标题、描述、SEO 标题、SEO 描述
- 每类同时支持：英文模板 + 中文注释
- 新增查看页面：`/settings/prompt-libraries`
- 详情页改写时可按“模板名”选择提示词库

## 5) 本轮文档产出

- `docs/PHASE2_PLAN.md`：第二阶段规划（范围、优先级、DoD）
- `docs/PROMPT_LIBRARY_ARCHIVE.md`：提示词完整归档
- `README.md`、`docs/REQUIREMENTS.md`：与当前实现对齐

## 6) 建议下一步（短期）

- 把 `vendor / 价格系数 / SKU 规则` 改成“店铺级可配置”
- 给提示词库增加“新增/编辑 UI”（目前以 JSON 管理为主）
- 补一组回归测试：编辑发布、AI 改写、多模板切换、eBay SKU 提取

## 7) 2026-03-26 增量

- 首页新增批量输入、50 条分页、列表进度条、缩略图占位和操作区优化
- 重复 ASIN 提交改为“刷新更新时间，不自动重抓”
- 导航升级为一级/二级分层，店铺设置与提示词库统一入口
- 详情页二次编辑区重排，发布配置与发布动作合并
- AI 改写结果持久化并在下次进入时默认回填，页面增加“已 ChatGPT 改写”提示
- 新增 `UPC维护` 页面：批量录入、12 位校验、已使用/未使用追踪
- Shopify 发布支持“创建后更新”：首次创建，后续更新同一商品
- 元字段升级为创建/更新后使用 GraphQL `metafieldsSet` 写入，富文本字段支持粘贴 HTML

## 8) 2026-03-26 增量（稳定性与排障）

- 元字段新增 `vehicle_fitment`、`package_list`，并与现有富文本字段统一处理链路
- 富文本转换增强：支持加粗/斜体/列表/H 标签（`<h1>`~`<h6>`）双向转换
- 已发布商品详情页增加自动同步 Shopify 数据，新增“同步 Shopify”手动按钮
- `metafieldsSet` 增加 `INVALID_TYPE` 自适应重试（按 Shopify 返回的期望类型修正）
- `warehouse`、`delivery_time` 在 GraphQL 后增加 REST 兜底 upsert，提升后台可见性一致性
- 修复详情页 `DetachedInstanceError`：模板上下文统一改为普通字典，避免 session 游离对象渲染失败

## 9) 2026-03-26 增量（运营体验与UI统一）

- 提示词库从只读升级为可维护：支持新增/修改/删除，模板 ID 改为系统自动生成并固定
- 改写模板兼容增强：未知占位符不再触发 500，兼容 `{title}` / `{description}` 历史写法
- 详情页同步规则调整：进入页和刷新仅读本地草稿；仅点击“从shopify同步产品信息”才触发同步
- 商品描述编辑区支持“HTML源码 / 可视预览”切换，默认可视预览，编辑高度提升
- SEO 标题/描述新增实时字符计数显示，并在超限时醒目提示
- 元字段富文本工具栏补齐 `H1~H5`
- 列表页新增 Shopify 状态列；详情页顶部增加醒目“已同步到 Shopify / 未同步到 Shopify”状态标识
- 进行全站 UI 一致性收敛：统一提示条、按钮语义色、表单宽度和通用布局类
